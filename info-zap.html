Design Overhaul

SocialTap is a social media links page manager. They asked me for help suggesting possible improvements to their website's design. I stripped the entire site, keeping only the important assets, in order to help the customer better understand the service SocialTap provides. The new design uses Javascript, GSAP animations and SASS to deliver a polished website. HTML5 <video> best practices are also used in order to prevent bothersome UX issues.

javascript,  GSAP,   sass,   HTML5,   netlify

Mobile Responsive

The biggest problem with the old design was how unfriendly it was on mobile devices. The menu opened and broke on smaller screen sizes which made navigating the site difficult. A lot of information would pile together and become hard to read when the screen changed size. I fixed this issue by resizing all elements to fit any screen size.
