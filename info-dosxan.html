Mobile-First

The influx of many new devices, and screen sizes requires us as developers to learn to learn about the importance of a mobile-first approach. Mobile-first means that I start the development and design process with the smallest screens in mind and work my way up to larger screen sizes. By giving mobile design priority, I am ensuring a optimal experience for my client's customers. A mobile first approach also helps keep my code organized. By prioritizing smaller screens first, I am forced to develop core site functionality first and progressively add on to it with the given increase in screen size.

Tools & Tech

When building Dosxan Digital, I wanted to create a site that delivered a well ordered, and minimalistic design. I also kept the tools I used to a minimum. I styled the site using CSS3 and flexbox. All functionality was added using pure Javascript. I really enjoy the opportunity to be able to build something with only Javascript as I feel it helps me to further cement core fundimental developer principles. All of the forms in Dosxan Digital use Javascript to ensure everything has been properlly filled out. This prevents empty forms, or errors from being sent back to our server.
